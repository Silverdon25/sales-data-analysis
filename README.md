# Sales Data Analysis Dashboard

## Project Overview
This project analyses sales data to identify trends, top-performing products, and regional performance.

## Key Features
- Data cleaning and exploration using Pandas
- Sales analysis by region
- Top-performing product analysis
- Profitability analysis
- Data visualisation using Matplotlib

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- The West region generated the highest sales
- A small number of products contribute most of the revenue
- Profit varies significantly across regions

## How to Run
1. Open the notebook in Jupyter
2. Run all cells
3. View analysis and visualisations
